# Система онлайн-заказа пиццы

Разработка системы для автоматизации процессов обслуживания клиентов пиццерии с поддержкой нескольких интерфейсов.

## Интерфейсы системы
- **Веб-сайт**
- **Мобильное приложение**
- **Telegram-бот**

## Основной функционал

### Для клиентов:
- Регистрация и авторизация
- Просмотр меню с ценами
- Формирование заказа (пицца, напитки, добавки)
- Выбор доставки или самовывоза
- Оплата онлайн или при получении
- Отслеживание статуса заказа

### Для администратора:
- Управление меню (добавление, изменение, удаление)
- Управление ценами и акциями
- Просмотр и управление заказами
- Изменение статусов заказов

## Технологии
- **Backend**: FastAPI
- **База данных**: PostgreSQL
- **Мобильное приложение**: Kotlin
- **Telegram-бот**: Aiogram
